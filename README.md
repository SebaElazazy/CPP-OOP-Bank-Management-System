# 🏦 C++ OOP Bank Management System

A complete Console-Based Bank Management System developed in C++ using Object-Oriented Programming (OOP) principles.

This project simulates a real banking system where administrators can manage clients, users, transactions, and currency exchange operations through a secure and user-friendly console interface.

---

# 📌 Overview

The application was developed to practice advanced C++ programming concepts and build a real-world project using Object-Oriented Programming.

The project follows a modular design where every feature is separated into dedicated classes and screens, making the code organized, reusable, and easy to maintain.

---

# ✨ Features

## 🔐 Login System

- Secure user authentication
- Username & password validation
- Login history (Login Register)
- Invalid login detection

---

## 👥 Client Management

- Add new client
- Delete client
- Update client information
- Find client
- List all clients

## 👨‍💼 User Management

- Find user

---

## 💰 Transactions

- Deposit

- Total balances

---

## 💱 Currency Exchange

- List all currencies
- Find currency
- Update exchange rates
- Currency calculator
- Currency converter

---

## 🔑 User Permissions

The system uses Bitwise Permissions to control access to different system features.

Permissions include:

- Show Clients
- Add Client
- Delete Client
- Update Client
- Find Client
- Transactions
- Manage Users
- Currency Exchange
- Full Access
